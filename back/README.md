# BTG Pactual - Backend

API REST en Node.js/Express para gestionar suscripciones a fondos, cancelaciones, historial transaccional y notificaciones SNS.

## Documentacion corta
- Guia funcional y despliegue: `docs/guia-rapida.md`

## Quick start local
```bash
cd back
cp .env.example .env
npm install
npm run dev
```
